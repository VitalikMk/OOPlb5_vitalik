Лабораторна №5

Инструкции
Створіть клас Товар, який містить інформацію про назву, ціну, опис, категорію тощо.
Реалізуйте клас Користувач, з атрибутами логіна, пароля, історії покупок.
Створіть клас Замовлення з деталями про товари, кількість, загальну вартість та статус замовлення.
Впровадьте інтерфейс ISearchable, який дозволить здійснювати пошук товарів за різними критеріями (ціна, категорія, рейтинг).
Реалізуйте клас Магазин, який забезпечує управління користувачами, товарами та замовленнями, а також імплементує пошуковий інтерфейс на рівні консольного додатку.

Laboratory №5

Instructions.
Create a Product class that contains information about the name, price, description, category, etc.
Implement the User class with login, password, and purchase history attributes.
Create the Order class with details about the products, quantity, total cost, and order status.
Implement the ISearchable interface that will allow you to search for products by various criteria (price, category, rating).
Implement the Store class, which provides management of users, products, and orders, as well as implements a search interface at the console application level.
